# Description

Implementation of face-api.js

## Installation

```bash
$ npm install
```

## Runing the code

```bash
# face detection
$ ts-node src/faceDetection.ts

# face detection + 68 landmark point + face descriptor (128 encoding)
$ ts-node src/faceLandmarkDetection.ts

# face recognition
$ ts-node src/faceRecognition.ts
```
